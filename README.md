# FBWG-Editor
Fireboy &amp; Watergirl Level Editor

## Object id's 
0 = Air
1 = Ground
2 = Top slope facing right
3 = Top slope facing left
4 = Bottom slope facing left
5 = Bottom slope facing right
6 = Water
7 = Lava
8 = Green stuff

## Color id's
1 = red
2 = green
3 = blue
4 = yellow 
5 = pink
6 = light blue
7 = purple
8 = white
